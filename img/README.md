# Imágenes

Las fotografías utilizadas en este proyecto se cargan directamente
desde **Unsplash CDN** mediante URLs externas.

No se almacenan imágenes localmente en este repositorio.

Todas las imágenes de Unsplash se distribuyen bajo la
[Unsplash License](https://unsplash.com/license), que permite
su uso gratuito con fines comerciales y no comerciales.
